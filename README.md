# NEWM-N328-Final-Project

This repository contains the packaged D3 anime dashboard.

## Contents
- `anime-d3-viz.zip`: the full static demo bundle

## Run locally
1. Unzip `anime-d3-viz.zip`
2. Open the `anime-d3-viz` folder
3. Start a local server, for example:

```powershell
python -m http.server 8000
```

4. Open `http://localhost:8000` in your browser

## Notes
- The demo uses D3 in the browser.
- Serving over HTTP is recommended so fetch requests work consistently.
